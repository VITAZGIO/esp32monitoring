# ESP32 Monitoring Panel

ESP32-панель мониторинга для домашнего сервера с экраном TFT, MQTT, Home Assistant, RF433, управлением реле, вентилятором, кнопками и датчиком температуры блока питания.

## Возможности

- отображение температуры, CPU и RAM для двух узлов:
  - Orange Pi
  - Proxmox
- MQTT-интеграция с Home Assistant
- включение и выключение подсветки экрана
- управление 4 реле
- управление PWM-вентилятором
- 12 кнопок с публикацией обычного и двойного нажатия
- RF433:
  - передача кодов
  - приём кодов
  - фильтрация trigger-кодов и lights-кодов
- буззер с готовыми звуковыми последовательностями
- датчик температуры БП на DS18B20
- авто-discovery датчика БП в Home Assistant

## Основные функции проекта

### Экран TFT
На экране отображаются:

- статус Wi-Fi
- статус Home Assistant
- Orange Pi:
  - температура
  - CPU
  - RAM
- Proxmox:
  - температура
  - CPU
  - RAM

### Подсветка экрана
Поддерживается управление подсветкой через MQTT.

### PWM-вентилятор
Есть 10 уровней скорости вентилятора.

Логика инверсная:

- чем меньше PWM, тем выше скорость
- `1` = минимальная скорость
- `10` = максимальная скорость

### Реле
Поддерживается 4 реле:

- ON
- OFF
- TOGGLE
- PULSE
- `PULSE_XXX`

### Кнопки
Подключено 12 кнопок:

- 1..6 через два ADC входа
- 7..12 через резистивную лестницу на GPIO39

Публикуется:

- обычное нажатие
- двойное нажатие

### RF433
Поддерживается:

- передача кодов
- удержание отдельного кода `hold97`
- приём кодов
- разделение кодов на:
  - trigger
  - lights
  - other

### Буззер
Поддерживаются команды:

- `STOP`
- `MARIO`
- `STARWARS`
- `BEEP_2500`
- `SOFT_800`
- `ERROR_600`
- `NOTIFY`
- `BOOT`
- `WARN`
- `SUCCESS`

### Датчик температуры БП
Используется DS18B20.

Публикуется в MQTT и автоматически добавляется в Home Assistant через discovery.
